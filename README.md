# CSSArtFestival
A CSS art project that recreates Pikachu using only HTML and CSS. Built without images or SVGs, it showcases creative use of CSS shapes, gradients, and positioning to create a detailed character design.

A CSS art project that recreates **Pikachu** using only HTML and CSS. The design is hand-crafted with `div` elements and CSS properties such as `border-radius`, `transform`, `box-shadow`, gradients, and absolute positioning to accurately capture Pikachu's iconic appearance without using images, SVGs, or external graphics. The project demonstrates creativity, precision, and advanced CSS styling techniques.
